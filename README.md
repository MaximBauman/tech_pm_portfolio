# Portfolio page

Статическая веб-страница портфолио для Максима Баумана с интерактивным экраном проектов,
full-screen переходом в кейс и live screencast-превью.

## Запуск

Можно открыть `index.html` напрямую в браузере. Для локальной проверки через сервер:

```bash
python3 -m http.server 4173
```

Страница не использует внешние зависимости и работает офлайн. Видео и иконки лежат в `assets/`.

## Где править контент

- Основной профиль: `index.html`
- Проекты, стек, описания, координаты и ассеты: массив `projects` в `script.js`
- Визуальный стиль, portal-переход и адаптивность: `styles.css`
